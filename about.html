About Credit Profile
System

A college project exploring AI-driven credit scoring based on e-commerce purchasing behavior, built with a modern full-stack architecture.

Project Overview

Traditional credit systems rely on bank statements and loan history — data that many young consumers simply don't have. This project explores an alternative: using e-commerce behavioral data to build credit profiles.

By analyzing order frequency, payment preferences (prepaid vs COD), and return rates, the system generates a credit score between 300–900 and categorizes users into Low, Medium, or High risk groups — with personalized improvement suggestions.

Scoring Logic

Transparent and explainable credit calculation rules

ACTION SCORE IMPACT
Base Score (starting point) 500
Each order placed +2 points
Each prepaid order (UPI / Card) +5 points
Each COD (Cash on Delivery) order −5 points
Each item returned −10 points
Minimum score limit 300
Maximum score limit 900
